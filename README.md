# 🎭 Театральный Генератор Постов

Flask-приложение для создания анонсов театральных мероприятий.  
Посты генерируются языковой моделью через **ProxyAPI** (поддержка GPT, Claude, Gemini) с возможностью экспорта в PDF.

## Возможности

- **LLM-генерация** — уникальные тексты через ProxyAPI (модель настраивается в `voice.md`)
- **Резервный шаблон** — если API-ключ не указан, работает шаблонный генератор
- **Экспорт в PDF** — через ReportLab с поддержкой кириллицы
- **Настройка голоса** — тон, длина, эмодзи, стиль заголовка (файл `voice.md`)
- **Тёмный интерфейс** — адаптивная вёрстка, спокойные цвета

## Установка и запуск

```bash
pip install -r requirements.txt
python app.py
```

Открой `http://127.0.0.1:5000`

## Настройка API

Для работы LLM-генерации создайте файл `.env` в корне проекта:

```
API_KEY=your_proxyapi_key_here
```

Ключ можно получить на [proxyapi.ru](https://proxyapi.ru).  
Если ключ не указан, приложение использует шаблонный генератор.

## Настройка голоса (`voice.md`)

```yaml
общий_тон: уютный, вдохновляющий, театральный
длина_поста: длинная
количество_эмодзи: 2
стиль_заголовка: восклицательный
дополнительные_указания: использовать театральные метафоры
модель: gpt-5.4-mini
```

Параметр `модель` — любая модель из списка доступных через ProxyAPI.
